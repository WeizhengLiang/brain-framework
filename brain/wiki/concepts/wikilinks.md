# Wikilinks

## Definition
Double-bracketed references (`[[page-name]]`) used throughout the brain to connect pages. They are the primary navigation mechanism in Obsidian's graph view and make the knowledge base traversable by both humans and agents.

## Key Points
- Core rule: use `[[wikilinks]]` everywhere — every page should link to related pages
- Wikilinks work across the entire vault, so brain pages can reference user notes outside `brain/`
- Agents use wikilinks to discover related context without loading everything into the [[context-window]]
- Orphan pages (no incoming or outgoing links) indicate missing connections — the wiki-writing skill flags these

## Related
